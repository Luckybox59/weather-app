# Консольное погодное приложение

Простое консольное приложение для получения актуальной информации о погоде с использованием API [OpenWeatherMap](https://openweathermap.org/).

## Возможности

-   Получение погоды по названию города.
-   Получение погоды по географическим координатам (широта и долгота).
-   Кэширование запросов для снижения нагрузки на API и работы при отсутствии сети.
-   Автоматические повторные запросы при временных сбоях сети.
